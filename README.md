# Natalia Kubala Portfolio

This repository showcases selected AI-assisted tools and workflow applications focused on analytics, document workflows, reporting, and IBM Jazz-integrated quality processes.

The projects were built for real enterprise use in an automotive engineering environment. This public version is intentionally anonymized: company-specific names, internal hosts, personal data, runtime files, and credentials were removed or replaced with placeholders. IBM Jazz remains visible as a relevant technology reference.

## Projects

- `projects/defects-findings-lessons-learned`
  - Defect and finding analysis tool with repeated-case grouping, Lessons Learned candidate creation, IBM Jazz integration, and PDF export.
- `projects/standards-reference-checker`
  - Standards detection and verification tool for PDF, DOC, and DOCX files with SVN-based repository checks.
- `projects/ll-cases-review-assistant`
  - Review workflow application for XLSX and CSV data with IBM Jazz integration and AI-assisted coverage analysis.
- `projects/report-checker`
  - PDF validation tool for functional test reports with automated rule checking and streamed analysis results.
- `projects/charging-log-analytics`
  - Charging-log analytics application with anomaly review, ranking snapshots, and scheduled background processing.
- `projects/reqif-pdf-anonymizer-supporting`
  - Supporting document anonymization tool for PDF and ReqIF-style exports.

## Note

Some projects depend on private infrastructure, internal datasets, or enterprise integrations, so local startup may require your own environment variables, sample data, and replacement services.
